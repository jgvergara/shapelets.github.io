About Shapelets


Shapelets is an open-source platform created for time-series analysis. An open framework that empower users to leverage analytical capabilities in big data environments.

Shapelets provides a handful of algorithms for time-series analysis coming up from the latest research in academia including our Khiva library

A platform that goes beyong anomaly detection offering more powerful analysis like motif and discord discovery, semantic segmentation, clustering, classification, rule discovery, etc..
